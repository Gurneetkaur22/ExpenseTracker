# ExpenseTracker
ExpenseTracker is a personal finance management application that helps users track their incomes and expenses, generate reports, and gain insights into their spending habits. It is designed to provide a simple, user-friendly interface with file-based persistence.

# Features:
1. Registeration for users
2. Login for users
3. Add income and expenses
4. View Expenses
5. View Category Wise Analysis
6. View Weekly Reports of Expenses
7. View Monthly Reports of Expenses


File-based persistence (saves data locally) using hibernate

#Tech Stack:
Language: Java
IDE: Eclipse
Version Control: Git & GitHub
Testing: Postman / Web Browser
